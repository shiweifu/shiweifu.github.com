使用vim和wget批量下载文件

2012-09-10 by shiweifu

遇到连续链接的下载地址，类似这样的：
http://abc.com/file[1-20]
怎么办？ Windows 下的主流下载工具都支持，呵呵，但其不在本文讨论范围内。
本文讨论的方法是使用vim来生成有序的链接地址，然后丢给wget/curl 这类下载工具来进行处理。

vim 支持递增数字：
ctrl+a

vim 支持录制宏：
q[a-z]

播放宏：
@[a-z]

wget 支持从文件加载下载地址：
-i, --input-file=FILE 下载本地或外部 FILE 中的 URLs。

齐活儿。

抛砖引玉：

  • 创建个文件，贴上文件地址
  • 开始录制宏
    • 对进行变化的数字＋1
    • 换行
  • 结束录制宏，再按下q
  • 开始播放宏，想要多少个链接要多少个链接
  • wget -i filename
  • 收工

引用某论坛看到的一句话：
照这个思路，其实你啥都不缺
